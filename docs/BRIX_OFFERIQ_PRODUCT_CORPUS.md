# BRIX OfferIQ Product Corpus v1.0

## Purpose

OfferIQ is the transaction intelligence engine of BRIX.

Its purpose is to transform acquisition analysis into acquisition action. OfferIQ converts DealIQ intelligence into professional documents, communications, offers, negotiations, and transaction workflows.

OfferIQ eliminates duplicate data entry and ensures transaction documents are generated directly from DealIQ data.

## User Goal

OfferIQ answers: **How do I pursue this opportunity?**

Users should move seamlessly from analysis to execution. OfferIQ should reduce transaction friction, improve consistency, and accelerate acquisitions.

## Core Principle

If DealIQ determines a property is worth pursuing, OfferIQ determines what should happen next.

OfferIQ should never require users to re-enter information already known by BRIX.

## Workflow

1. DealIQ Analysis
2. Acquisition Recommendation
3. User selects Pursue Opportunity
4. OfferIQ Activated
5. Documents Generated
6. Communications Generated
7. Negotiations Managed
8. Contract Tracking
9. PipelineIQ Updated

After contract execution, OfferIQ should support:

- Inspection Tracking
- Title Review
- Insurance Review
- Financing Progress
- Appraisal Tracking
- Closing Requirements
- Document Collection

## Transaction Timeline

Display key dates, deadlines, contract milestones, inspection dates, financing milestones, and closing date.

## Integration Requirements

OfferIQ consumes data from:

- FindIQ for opportunity source
- DealIQ for acquisition intelligence
- PipelineIQ for workflow status
- PortfolioIQ for closed asset creation

OfferIQ acts as the execution layer between underwriting and ownership.

## Automation

Automation should eliminate repetitive work.

Examples:

- Generate offer package from DealIQ
- Create negotiation summary automatically
- Create due diligence checklist automatically
- Notify users of approaching deadlines
- Generate closing package automatically

## Success Metric

OfferIQ succeeds when users can answer:

**What is the best way to pursue this opportunity, and what should happen next?**
